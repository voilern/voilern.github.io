一些例题

  1. 根据声明int a;表达式(a&3)==(a%4)的值为？
    答案：1
    解析：均为取a的最后两位
  2. 数组定义为int a[3][2]={1,2,3,4,5,6},数组元素____的值为6？
    答案：a[1][3]
    解析：a[1][3]相当于从a[1][0]开始向数组后移三位，即得到6
    可以画一个3行2列的数组后递推选项所指向的位置
  3. int x[] = {1,2,3}, *p = x, y;则y = *p++相当于？
    答案：y = x[0], p++;
    解析：p++的结果为p操作后再自增
  4. int p[5], * a[5],表达式正确的是？
    答案：*(a+1) = p;
    解析：指针赋指针
